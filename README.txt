# 📂 File Organizer (Python)

A simple Python script that automatically organizes files in a folder into categories like Images, Videos, and Others.

---

## 🚀 Features
- Organizes files automatically
- Supports Images, Videos, and Other files
- Creates folders if they don’t exist
- Prevents duplicate file overwrites

---

## 🛠️ Technologies Used
- Python
- os module
- shutil module
